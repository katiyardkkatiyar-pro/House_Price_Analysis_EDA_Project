## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on a Housing Dataset containing 2,000 property records. The goal is to identify the factors that influence house prices and generate actionable insights through data visualization and feature engineering.

## 🛠 Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Dataset Features

* Area
* Bedrooms
* Bathrooms
* Floors
* YearBuilt
* Location
* Condition
* Garage
* Price

## 🔍 Analysis Performed

### Data Exploration

* Dataset Structure

* Statistical Summary

### Univariate Analysis

* Price Distribution
* Location Distribution

* Garage Analysis
* Outlier Detection

### Bivariate Analysis

* Area vs Price
* Location vs Price

* Bathrooms vs Price

### Feature Engineering

Created new features:

* Age
* Price_per_Sqft
* Total_Rooms

### Multivariate Analysis

* Correlation Heatmap
* Pairplot
* Dashboard Visualization

## 📈 Key Insights

* House area has a strong positive impact on price.
* Properties in premium locations have higher average prices.
* Houses with more bedrooms and bathrooms generally cost more.
* Price outliers exist and may represent luxury properties.
* Price per square foot varies across locations and conditions.
